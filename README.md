# .NET-Video-Converter
Video Converter. Convert any video files (such as *.mpg; *.mp4; *.webm; *.wmv; *.asf; *.mov; *.avi; *.mkv; e.t.c) to popular video and audio formats like MP3, MP4, AVI, e.t.c with the highest quality possible.Uses FFMPEG.
